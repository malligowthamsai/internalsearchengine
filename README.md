# 🔍 Mini Content Search Website

A simple prototype of a **searchable content website** where users can type a query and find matching articles from a small dataset.

---

## 🚀 Project Overview

This project demonstrates a basic **internal search engine** built using simple web technologies.

Instead of searching the internet, users can:

* Enter a query in the search bar
* Instantly get matching results from stored content
* Click and read the relevant article

---

## 💡 Features

* 🔎 Search bar for user queries
* 📄 Predefined articles/content
* ⚡ Instant filtering based on keywords
* 🧠 Simple search logic (no external APIs)
* 💻 Fully client-side (no backend required)

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

---

## 📂 Project Structure

```
/project-folder
│── index.html      # Main UI
│── style.css       # Styling
│── script.js       # Search logic
│── data.js         # Content/articles
```

---

## ⚙️ How It Works

1. Content is stored as a list of objects (title + description).
2. User types in the search bar.
3. JavaScript filters content based on keywords.
4. Matching results are displayed instantly.
